# Studying Web-Development
Follow my journey of studying Web-Development in Germany. 

## Check it out

Simply clone the directories and files using the git built-in cloning-tool.

```bash
git clone https://github.com/Maretyui/study.git
```

## Usage

After installing all the files, start it either by simply opening the ``index.html`` file in the browser,  
or by starting up ``Live Server`` in the root directory.
